## 什么是 Flowise？

Flowise 是一款开源低代码工具，供开发者构建定制化的 LLM 编排流程和 AI 智能体。

开发 LLM 应用通常涉及无数次的迭代。我们的低代码和拖放式 UI 方法能够实现快速迭代，帮助您更快地从测试阶段过渡到生产阶段。
