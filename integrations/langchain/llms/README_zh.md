# 大型语言模型 (LLM)

***

大型语言模型 (简称 LLM) 是一种在海量文本数据上训练的 AI 系统。这使其能够进行沟通并生成类似人类的文本以响应各种提示和问题。本质上，它们能够理解和回应复杂的语言。
