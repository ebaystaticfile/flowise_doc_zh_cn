## 输入

| 参数      | 描述                                                                       | 默认值       |
| -------- | --------------------------------------------------------------------------- | ------------- |
| 会话 ID    | 用于检索/存储消息的 ID。如果未指定，则将使用随机 ID。                         |               |
| 内存键    | 用于在提示模板中格式化消息的键                                             | `chat_history` |
