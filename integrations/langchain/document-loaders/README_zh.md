# 文档加载器

***

文档加载器允许您从不同的来源加载文档，例如PDF、TXT、CSV、Notion、Confluence等。它们通常与[向量存储](../vector-stores/)一起使用，以嵌入的形式进行上载，然后可以在查询时检索。
