# 聊天模型

***

聊天模型接收一系列消息作为输入，并返回模型生成的消息作为输出。这些模型（如 **gpt-3.5-turbo** 或 **gpt4**）功能强大，且比其前身 Completions 模型（如 **text-davincii-003**）更便宜。
